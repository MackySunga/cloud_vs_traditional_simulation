What students should notice

When demand changes quickly, cloud computing usually responds faster because resources can scale on demand. Traditional computing works well for predictable workloads, but sudden spikes can create queues while new hardware is being added.

Interpretation

Start the simulation, increase spike intensity, and watch the traditional queue grow first. Then turn off cloud autoscaling to show that cloud advantages depend on proper configuration, not just the word "cloud."
